## 🎹 Shortcuts Guide

### 🖥 MacOS
- `esc` — **Close app**
- `⌘ (Left CMD)` + `⌘ (Right CMD)` — **Pause/Unpause**
- `s` — **Show stats** (only when paused)
- `t` — **Open `text.txt`** (only when paused)

### 🖥 Windows
- `esc` — **Close app**
- `Left alt` + `Right alt` — **Pause/Unpause**
- `s` — **Show stats** (only when paused)
- `t` — **Open `text.txt`** (only when paused)
